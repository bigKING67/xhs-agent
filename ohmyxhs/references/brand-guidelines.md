# 小红书品类指导与风险规则

## 品类差异化策略详解

### 【美妆类】

**核心策略**：强调"修饰效果"和"快速见效"，展示产品的变身潜力

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 妆容灵感（通勤妆、约会妆）→ 产品展示 → 完成妆容效果 |
| **可信度增强** | 展示上妆过程、分享妆效对比、标注气质标签、提及持久度 |
| **禁忌场景** | 避免与医疗混淆、避免针对特殊肤质问题的功效声称 |
| **标题示例** | "通勤妆 \| 这个口红让我气质大升" |

**禁用词**：修复、激活、祛、治疗、医疗暗示词

**高风险词**：变白（需"可能显白"）、显脸小（需对比）、气质提升（过于主观）

---

### 【护肤类】

**核心策略**：强调"真实测评"而非"一次见效"，建立信任比追求完美更重要

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 痛点场景（黑眼圈、油皮困扰）→ 亲测对比 → 真诚建议 |
| **可信度增强** | 展示使用过程、分享细节体验、标注时间周期（如3周）、提及可能出现的过渡期 |
| **禁忌场景** | 避免在医学语境中讨论、避免与医疗产品混淆、避免针对疾病类皮肤问题 |
| **标题示例** | "眼膜 \| 黑眼圈困扰 3周逆袭记，终于不用遮瑕了" |

**禁用词**：绝对词（最好/第一）、医疗暗示词（修复/再生/激活）、永久类词

**高风险词**：淡化、改善、提亮（需加限定词）

---

### 【3C / 电子产品类】

**核心策略**：强调"使用场景"和"生活改变"而非参数堆砌，讲故事比列数据更吸引人

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 痛点场景（办公效率、出差便携、游戏体验）→ 实际体验展示 → 适用人群明确 |
| **可信度增强** | 展示真实使用细节、对比体验差异、提及学习曲线、标注价格/性价比 |
| **禁忌场景** | 避免与竞品直接对标、避免技术参数虚假、避免承诺保修/质保 |
| **标题示例** | "出差必备 \| 这个小工具让我工作效率翻倍" |

**禁用词**：颠覆、革命性、行业首创、绝对性能词

**高风险词**：最快、最强、最轻（需对比基准）、省电（需技术支撑）

---

### 【食品 / 饮品类】

**核心策略**：强调"体验与氛围"而非功效声称，讲品尝体验比讲营养价值更有吸引力

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 场景营造（下午茶、懒人食品、聚会必备）→ 味觉细节描写 → 心情分享 |
| **可信度增强** | 展示拆箱、品尝过程、配餐建议、分享心情感受、标注原产地/品牌故事 |
| **禁忌场景** | 避免任何健康功效声称、避免与药物或医学概念混淆 |
| **标题示例** | "下午茶 \| 发现一款治愈系零食，甜到心底" |

**禁用词**：医疗功效词（增免疫、改善体质、调理体质）、绝对性营养词

**高风险词**：健康、排毒、代谢、养生（这类词都与医疗有关联，极易违规）

---

### 【生活用品 / 家居类】

**核心策略**：强调"生活品质提升"和"实用性"，讲解决生活痛点比讲产品功能更实用

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 生活痛点（收纳困难、清洁耗时、空间浪费）→ 真实解决方案展示 → 值得推荐指数 |
| **可信度增强** | 展示使用前后对比、分享长期使用体验、提及适用场景、分享家居布置灵感 |
| **禁忌场景** | 避免环保/健康/安全类大声明、避免医学/卫生防疫相关词汇 |
| **标题示例** | "收纳 \| 衣柜再小，这个神器也能帮你多收50件衣服" |

**禁用词**：夸大功效词（完全、彻底、永久）、医疗相关词

**高风险词**：健康、安全、无害（需认证支持）、环保（需证书）、天然（需验证）

---

### 【教育 / 课程类】

**核心策略**：强调"学习成果"和"投资回报率"，讲个人成长比讲课程内容更有吸引力

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 学习困扰（"我也不会编程，但…"）→ 课程亮点与学习过程 → 个人成长与成果展示 |
| **可信度增强** | 分享学员真实反馈、展示作品或成果、提及实操性（"边学边做"）、标注适用人群 |
| **禁忌场景** | 避免保证学员就业、避免虚假成功案例、避免对标大牌讲师 |
| **标题示例** | "0基础学摄影 \| 3个月从小白到能接单" |

**禁用词**：绝对词（最好、必学、唯一）、过度承诺词（年赚100万、保证通过）

**高风险词**：改变人生、彻底改变、保证、名师推荐、零基础包学会

---

### 【旅游 / 体验类】

**核心策略**：强调"沉浸式体验"和"值得去的理由"，讲故事比讲景点更能吸引人

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 旅游痛点或憧憬（"想逃离城市"、"寻找小众景点"）→ 行程体验展示 → 旅游攻略与建议 |
| **可信度增强** | 展示实地拍摄的照片/视频、分享真实花费和体验感受、提供具体行程建议、标注最佳季节/人数 |
| **禁忌场景** | 避免虚假照片（修图过度）、避免隐瞒消费陷阱、避免对标其他景点的贬低 |
| **标题示例** | "苏州古镇 \| 终于找到不拥挤的江南梦" |

**禁用词**：绝对词（世界最美、一生必去）、过度营销词（天堂、仙境）

**高风险词**：最便宜、最划算（需数据对比）、绝对不踩坑（主观性强）

---

### 【母婴 / 儿童类】

**核心策略**：强调"安全性"和"宝宝舒适度"，父母最关心的是产品有没有问题、宝宝喜不喜欢

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 养育痛点（"宝宝总是拉肚子，我试过很多办法…"）→ 产品如何改善 → 真实使用反馈与建议 |
| **可信度增强** | 分享宝宝使用前后的具体变化、展示产品成分表或安全认证、提及使用方法和注意事项 |
| **禁忌场景** | 避免任何与健康、疾病相关的功效宣传、避免对标奶粉品牌、避免承诺"不过敏" |
| **标题示例** | "婴儿纸尿裤 \| 宝宝红屁股困扰终于解决了" |

**禁用词**：医疗类词（治疗、祛、消除）、医学级词（专业级、医用级）

**高风险词**：营养、增强体质、提高免疫（严重违规）、适合所有宝宝

---

### 【健身 / 运动类】

**核心策略**：强调"运动效果"和"身体感受"，讲个人蜕变比讲产品参数更有激励性

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 运动困扰（"我是完全的健身小白…"）→ 产品或服务如何帮助 → 个人运动成果与心得 |
| **可信度增强** | 展示运动过程和汗水（真实感）、分享身体感受的具体变化、提及坚持周期（如"3个月"） |
| **禁忌场景** | 避免医学语言（"改善体质""提高代谢"）、避免承诺快速效果、避免性别刻板印象 |
| **标题示例** | "居家瑜伽 \| 我是怎样在3个月内养成运动习惯的" |

**禁用词**：医疗词（治疗、康复、矫正、纠正）、绝对词（最强、必瘦）

**高风险词**：减肥（改为"运动"）、燃脂（需数据支持）、增肌（需科学论证）

---

### 【医疗设备 / 美容仪器类】⚠️ P0 新增 [参考案例4、6]

**核心策略**：强调"使用体验"和"非医疗属性"，明确区分产品功能（如"促进血液循环"）与医疗效果（如"治疗痘痘"）

> **最关键警示**：医疗设备类是高风险品类！必须严格避免任何医疗用语和功效承诺。产品越接近医疗功能（如美容仪、洁面仪、脱毛仪等），风险越高。

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 使用痛点（"皮肤护理很麻烦"、"想要提升肌肤状态"）→ 产品如何简化护肤 → 个人使用体验与感受 |
| **可信度增强** | 展示实际使用过程（操作展示）、分享皮肤的视觉变化（如"更光滑"、"看起来更明亮"）、提及持续使用时长（如"3周"）、强调个人体验而非绝对效果 |
| **禁忌场景** | **严禁医疗用语**（治疗、祛、修复、激活、再生）、避免声称能改善/治疗皮肤病、避免对标医疗手段、避免承诺立即见效、禁止宣传"医学级"或"医用级" |
| **标题示例** | "美容仪 \| 这个小仪器让我的日常护肤更高效了" |

**禁用词**（医疗设备专用扩展）：
- **绝对禁止**：治疗、祛、修复、激活、再生、治愈、康复、处方、医用、医学级、医疗级
- **医疗暗示词**：改善皮肤病、祛痘、祛斑、祛皱纹、祛黑头、治疗敏感肌、修复屏障、激活胶原蛋白
- **功效类**：永久、彻底、根治、完全消除、一次见效

**高风险词**（医疗设备必须加限定词）：
| 高风险词 | 违规原因 | 修改方案 |
|---------|--------|--------|
| 改善痘痘/粉刺 | 暗示医疗功能 | "使用过程中，我感觉皮肤状态有变化" 或 "很多用户反馈皮肤变得更清爽" |
| 清洁毛孔 | 夸大清洁能力 | "帮助深层清洁" 或 "可能有助于清洁肌肤" |
| 紧致提升 | 承诺医疗效果 | "使用后肌肤看起来更有弹性" 或 "坚持使用，肌肤可能变得更紧致" |
| 抗衰老/抗皱 | 暗示医疗抗衰 | "促进皮肤的健康状态" 或 "可能有助于维持肤质" |
| 祛黑头/粉刺 | 医疗词 | "深层清洁可能帮助改善黑头" 或 "使用后毛孔看起来更清爽" |

**诊断流程**：
1. 识别产品的实际功能（如"超声波振动"、"LED光照"）
2. 转化为非医疗的使用体验（如"使用更便捷"、"肌肤感受更好"）
3. 严格避免与医疗效果的任何关联
4. 如果无法转化，立即调整策略或拒绝宣传该功能

---

### 【医美服务 / 皮肤管理】⚠️⚠️ P1 新增 [参考案例5]

**核心策略**：强调"专业服务体验"和"安心感"，完全转化医疗属性为生活体验，用用户反馈而非医学声称

> **最关键警示**：医美服务是医疗+生活的最高风险融合品类！涉及医疗资质、功效承诺、医学用语等多个监管红线。严禁说"治疗"、"医学"、"有效率"、"医生"等任何医疗词汇。

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 护肤焦虑（"自己护肤踩坑"）→ 专业体验分享 → "放心感+收获感"的转变 |
| **可信度增强** | 强调师傅的诊断过程（不是医学，是美学诊断）、展示6次套餐的阶段进展（不是"疗程"而是"护理计划"）、分享真实体验和社群反馈 |
| **禁忌场景** | **严禁医疗用语**（治疗、祛、医学、医生、处方、有效率）、避免对标医疗手段、避免承诺立即见效、禁止声称"医学级"或"医用级" |
| **标题示例** | "医美初体验 \| 花6000块我终于明白什么叫专业护肤" |

**禁用词**（医美服务专用扩展）：
- **绝对禁止**：治疗、祛、医学、医生、医学级、医用级、有效率、临床证实、处方、康复
- **医疗暗示词**：改善皮肤病、祛斑、祛皱、祛痘、修复肌肤屏障、激活胶原蛋白
- **功效类**：永久、彻底、根治、完全消除、一次见效、包治愈

**安全替代表述**：
- ❌ "医生设计的治疗方案" → ✅ "师傅根据我的肤质诊断给出的定制护理计划"
- ❌ "这个疗程能有效改善斑点" → ✅ "很多用户反馈6次后皮肤看起来更均匀了"
- ❌ "医学级皮肤管理" → ✅ "专业级皮肤护理体验"
- ❌ "有85%有效率" → ✅ "很多人都感受到了皮肤的变化"

**医美服务专项风险流程**：
1. **识别医疗属性**：皮肤管理、光疗、导入等都涉及医疗
2. **转化为体验词**：不说"治疗"，改为说"护理体验"、"皮肤护理"
3. **强调过程和感受**：阶段进展、皮肤感受、社群反馈
4. **加上安全声明**：强调"个人体验"、"仅供参考"、"多数人反馈"

---

### 【金融理财 / 投资服务类】⚠️ P2 新增（高风险）

**核心策略**：强调"财商教育"和"理性决策"而非"收益承诺"，用真实理财故事传递"风险意识"和"长期思维"

> **关键警示**：金融服务是监管最严品类之一！涉及证券法、广告法、金融监管多重法规。严禁任何收益承诺、保本保收益、夸大投资回报。

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 理财困惑（"月入8千怎么存钱"）→ 学习过程分享 → 个人理财心得与成长 |
| **可信度增强** | 分享真实的理财学习经历、展示记账习惯和消费观变化、提及风险和踩坑经历（增加真实感）、标注"个人经验仅供参考" |
| **禁忌场景** | **严禁收益承诺**（保本、年化收益）、避免推荐具体股票/基金代码、避免冒充专业理财师（无资质）、禁止使用"稳赚""躺赚"等词 |
| **标题示例** | "存钱 \| 月入8千的我，如何1年攒下5万块" |

**禁用词**：
- **绝对禁止**：保本、保收益、稳赚不赔、零风险、躺赚、财务自由（承诺性）、年化XX%（承诺性）
- **金融违规词**：内幕消息、跟庄、庄家、割韭菜（贬义引导）、暴富、一夜暴富
- **夸大词**：最赚钱、最安全、收益最高、必涨、必赚

**高风险词**（需严格限定）：
| 高风险词 | 违规原因 | 修改方案 |
|---------|--------|--------|
| 理财收益 | 容易暗示承诺 | "我的理财学习心得"、"个人记账经验分享" |
| 推荐买XX | 投资建议 | "我个人选择了XX，但每个人情况不同" |
| 年赚XX万 | 收益承诺 | "通过记账和规划，我的存款有了变化" |
| 基金定投 | 具体投资建议 | "我在学习基金知识的过程中…"（强调学习） |

---

### 【房产 / 家居生活类】⚠️ P2 新增（高风险）

**核心策略**：强调"居住体验"和"生活方式"而非"投资升值"，用社区生活和装修体验传递"家的温度"

> **关键警示**：房产是高监管品类！涉及房产广告法、不动产交易规范。严禁承诺升值、学区保证、投资回报。

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 居住痛点（"租房vs买房的纠结"）→ 看房/装修经历分享 → 居住体验和生活方式展示 |
| **可信度增强** | 分享真实的看房经历和决策过程、展示装修前后对比、提及社区环境和邻居关系、标注"个人选择仅供参考" |
| **禁忌场景** | **严禁升值承诺**（必涨、投资回报）、避免标注具体楼盘价格走势、避免冒充房产中介或专业人士（无资质）、禁止使用"学区房保证""地铁房必涨"等词 |
| **标题示例** | "装修 \| 我的60平小家改造记，花5万打造温馨小窝" |

**禁用词**：
- **绝对禁止**：必涨、保值、投资回报率、升值空间大、学区保证、地段最好
- **房产违规词**：抄底、买到就是赚到、错过就没了、限时优惠（制造焦虑）
- **夸大词**：最便宜、性价比最高、最好的楼盘、绝对不亏

**高风险词**：
| 高风险词 | 违规原因 | 修改方案 |
|---------|--------|--------|
| 学区房 | 教育资源承诺 | "周边有XX学校，具体划片以官方为准" |
| 升值 | 投资承诺 | "我选择这里是因为生活便利"（强调居住而非投资） |
| 地铁盘 | 配套承诺 | "离地铁站步行约10分钟，通勤比较方便" |
| 首付XX万 | 价格引导 | "我当时的购房预算大概是…"（个人经验） |

### 【宠物用品 / 养宠生活类】P2 新增

**核心策略**：强调"毛孩子的快乐"和"铲屎官的省心"，用宠物日常和互动记录传递"有爱生活"

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 养宠困扰（"猫咪挑食怎么办"）→ 产品使用过程和宠物反应 → 铲屎官的真实心得 |
| **可信度增强** | 展示宠物使用产品的真实反应（视频/图片）、分享长期使用的观察记录、提及不同宠物品种的差异性、标注"我家毛孩子的情况" |
| **禁忌场景** | 避免宠物健康功效声称（治疗皮肤病、改善肠胃）、避免替代兽医建议、避免承诺"所有宠物都适用" |
| **标题示例** | "猫粮 \| 挑嘴猫终于不挑食了，这款粮我家主子居然吃光了" |

**禁用词**：
- **绝对禁止**：治疗、医疗级、兽医推荐（无证据）、根治、消除疾病
- **医疗暗示词**：改善皮肤病、治疗肠胃、消炎、杀菌（宠物用品）
- **夸大词**：所有宠物都爱、100%不过敏、永远不挑食

**高风险词**：
| 高风险词 | 违规原因 | 修改方案 |
|---------|--------|--------|
| 改善肠胃 | 医疗暗示 | "我家猫吃了之后便便变正常了"（个人观察） |
| 美毛 | 功效承诺 | "毛发看起来更有光泽了"（视觉描述） |
| 增强免疫 | 医疗暗示 | "猫咪精神状态不错，活蹦乱跳的" |
| 适合所有猫 | 绝对化 | "我家布偶猫很喜欢，但建议先少量尝试" |

### 【健康食品 / 功能性食品类】⚠️ P2 新增（高风险）

**核心策略**：强调"品尝体验"和"生活方式"而非"健康功效"，用日常饮食记录传递"吃得开心"

> **关键警示**：健康食品是高风险品类！与医疗、保健品边界模糊。严禁任何疾病治疗、体质改善、医疗功效声称。普通食品不得宣传保健功能。

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 饮食困扰（"想吃零食又怕胖"）→ 产品口感和体验分享 → 日常搭配和生活方式 |
| **可信度增强** | 展示产品外观、包装、配料表、分享口感和味觉体验细节、提及搭配食用建议、标注"个人口味仅供参考" |
| **禁忌场景** | **严禁健康功效**（改善体质、增强免疫、排毒养颜）、避免对标药品或保健品、避免声称"适合XX疾病人群"、禁止使用医学名词 |
| **标题示例** | "零食 \| 低卡零食合集，好吃不怕胖的小确幸" |

**禁用词**：
- **绝对禁止**：治疗、改善体质、增强免疫力、排毒、养颜、抗癌、降血压、降血糖
- **保健品暗示词**：调理身体、补气血、清肠、通便、助眠（功效声称）
- **夸大词**：最健康、纯天然无添加（需认证）、有机（需认证）

**高风险词**：
| 高风险词 | 违规原因 | 修改方案 |
|---------|--------|--------|
| 减肥/瘦身 | 功效承诺 | "低卡零食"、"控制热量的小零嘴" |
| 养生 | 医疗暗示 | "注重饮食搭配的生活方式" |
| 排毒 | 医疗词 | "清爽的口感"、"喝完感觉很舒服" |
| 美容养颜 | 功效承诺 | "好喝又好看的下午茶" |
| 无添加 | 需认证支持 | "配料表很简单"、"成分比较干净" |

### 【二手交易 / 闲置好物类】P2 新增

**核心策略**：强调"发现惊喜"和"物品新生命"，用淘货故事和改造过程传递"省钱又环保"的生活态度

| 维度 | 指导 |
|-----|------|
| **推荐角度** | 消费观转变（"大牌不一定要买新的"）→ 淘货过程和发现 → 二手好物的使用体验 |
| **可信度增强** | 展示实物图（多角度、细节瑕疵）、分享购买渠道和鉴别方法、提及物品真实成色和使用感受、标注"个人淘货经验" |
| **禁忌场景** | 避免保证正品（无鉴定）、避免隐瞒瑕疵、避免虚假比价（标高原价）、避免充当中间商（无资质） |
| **标题示例** | "二手 \| 500块淘到一只9成新大牌包，分享我的闲鱼捡漏经验" |

**禁用词**：
- **绝对禁止**：保证正品（无鉴定证书）、全新未拆（二手标注）、绝对不假
- **欺诈暗示词**：原价XX元（虚高标价）、限时甩卖、亏本出（制造紧迫感）
- **夸大词**：最值、必买、不买后悔、抄底价

**高风险词**：
| 高风险词 | 违规原因 | 修改方案 |
|---------|--------|--------|
| 正品保证 | 需鉴定支持 | "附带购买凭证"、"我在专柜验过" |
| 全新 | 二手不应标全新 | "9成新"、"几乎没使用过" |
| 原价XX元 | 虚高对比 | "官网参考价约XX元"（标注来源） |
| 捡漏 | 容易夸大 | "性价比不错"、"我觉得挺值的" |

## 全局禁用词清单

### 绝对化词汇（广告法明确禁止）

| 禁用词 | 原因 | 替代方案 |
|-------|------|--------|
| 最好、第一、唯一、顶级 | 无法验证，违反广告法 | 深受用户喜爱、热门推荐、很多人都选择 |
| 永久、彻底、根治 | 医疗暗示 | 明显改善、长期坚持有效果、持续使用更佳 |
| 100%有效、绝对有效 | 过度承诺 | 大多数用户反馈、据用户分享、亲测有效 |

### 医疗类暗示词（小红书严格禁止）

| 禁用词 | 原因 | 替代方案 |
|-------|------|--------|
| 修复、再生、活化、激活 | 医疗功效表述 | 改善、提亮、滋润、舒缓 |
| 祛、治疗、消除（黑眼圈/痘痘等） | 医疗功效 | 淡化、改善、调理、护理 |
| 排毒、代谢、净化、通透 | 医疗功效 | 清爽、清洁、疏通、清透 |

## 小红书平台规则要点

### CES 评分系统（关键词布局）

小红书推荐系统基于 CES 评分，综合考虑：
- **内容质量**：信息完整、写作流畅、美学价值
- **用户互动**：评论、收藏、转发数
- **合规性**：无违禁词、无医疗暗示

**最大化 CES 的方式**：
1. 关键词合理分布（标题 + 正文）
2. 避免一切违规词汇（会直接降权）
3. 建设互动氛围（CTA 引导评论/收藏）

### 搜索收录机制

- **核心关键词**优先权最高（在标题和前3句出现）
- **长尾关键词**精准度高，转化率高
- **热搜词**容易获得推荐流量（但竞争大）
- **品牌词**帮助建设品牌认知
