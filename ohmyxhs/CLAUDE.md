## 模块职责

ohmyxhs 是**小红书内容策略大师 AI Skill**，为 AI 智能体（Claude、Codex 等）提供结构化指导，帮助自动化生成：

- **种草笔记创作** — PAS 逻辑框架（痛点→分析→解决方案）
- **爆款标题生成** — 3 种标题方案（精准搜索式、解决方案式、悬念故事式）
- **合规性检查** — 禁用词汇、医疗暗示、夸大词、虚假背书
- **品类风险识别** — 15 大品类，其中 3 类超高风险（医美、金融、房产）
- **内容策略规划** — 洞察、人设、情绪、口吻、价值锚点
- **视觉封面设计** — 构图、色调、滤镜建议
- **算法优化建议** — CES 评分、关键词矩阵、发布策略

**核心特点**：
- **真诚第一** — 建立信任优于追求爆款
- **数据驱动** — 基于小红书推荐机制与用户心智
- **风险意识** — 熟悉平台规则，规避违规
- **多品类支持** — 15 大品类 + 跨品类指导

## 对外接口（AI 调用约定）

### 1. 三种输出模式

#### 完整模式 ⭐（推荐）
**适用**: 深度策划、预算充足、需要全套方案

**输出 5 大模块**（每模块详细说明）：
```
1. 策略概要 (Executive Summary)
   - 核心策略：2-3 句切入点
   - 预期效果：搜索流量、互动预测

2. 内容策略 (Content Strategy)
   - ① 核心洞察 — 用户困扰
   - ② 目标用户细分 — 年龄、职业、痛点、消费能力
   - ③ 主要唤醒情绪 — 获得感/安心感/自信感/共鸣感/掌控感
   - ④ 笔记主题与人设 — 主题/人设定位
   - ⑤ 博主演绎与口吻 — 闺蜜分享式/干货教程式/文艺分享式/故事叙述式
   - ⑥ 价值锚点 — 解决方案或启发

3. 标题与关键词
   - 3 个标题方案（精准搜索式、解决方案式、悬念故事式）
   - 关键词矩阵（核心/长尾/场景人群）

4. 视觉与封面设计
   - 构图建议（布局、主体位置、留白）
   - 色调与滤镜（日系/奶油色/电影感）
   - 核心元素（人物、产品、背景、道具）

5. 内容脚本
   - 图文正文（PAS 逻辑 + 结构化标题）
   - 或视频分镜（5 关键镜头）

6. 风险检查与规避建议
```

#### 精简模式 ⚡
**适用**: 快速验证、时间紧张、内容少于 3 篇

**输出**：
```
1. 策略概要 (简化版)
2. 内容策略 (核心要点)
3. 3 个标题方案 + 核心关键词
```

#### 极速模式 🔥
**适用**: 需要立即灵感、素材快速补充

**输出**：
```
- 一句话核心策略
- 3 个爆款标题
- 关键词组
```

### 2. 内容策略的 6 维度

**① 核心洞察** (Problem Insight)
```
示例: "熬夜党最怕的就是黑眼圈暴露年龄，却又没时间做复杂护理"
注意: 一句话点破用户在特定场景下的困扰或未被满足的需求
```

**② 目标用户细分** (Audience Segmentation)
```
示例: 25-35 岁都市职场女性，工作压力大、经常熬夜、看屏幕时间长。
      注重护理但时间紧张，消费能力中等偏上，审美偏向"素颜高级感"。
维度: 年龄、职业、生活场景、痛点、消费能力、审美偏好
```

**③ 主要唤醒情绪** (Emotional Triggers)
```
选项:
- 🌟 获得感 (Gain) — "买对了就赚到"
- 😌 安心感 (Safety) — "终于可以放心了"
- 💪 自信感 (Confidence) — "变得更自信"
- 👯 共鸣感 (Resonance) — "原来不是我一个人"
- 🎯 掌控感 (Control) — "我可以掌控这个"
```

**④ 笔记主题与人设** (Theme & Character)
```
示例:
  主题: "熬夜党黑眼圈自救指南"
  人设: "资深护肤学姐" 或 "懒人生活家"
定位: 选择最容易与用户建立信任的主题与身份
```

**⑤ 博主演绎与口吻** (Tone & Style)
```
4 种类型:
- 🗣️ 闺蜜分享式 — 亲切、随意、像在和闺蜜聊天
- 📚 干货教程式 — 专业、有条理、像在上课
- 🌿 文艺分享式 — 温暖、有审美、像在读日记
- 💬 故事叙述式 — 代入感强、像在讲故事
```

**⑥ 价值锚点** (Value Anchor)
```
不仅是产品功能，而是解决方案或启发。这是和其他笔记的差异化所在。

示例:
  不是: "这款眼膜补水效果很好"
  而是: "教用户黑眼圈形成的根本原因和日常预防方法，
        帮助从根本上减少困扰"
```

### 3. 标题方案模板

#### 方案 A — 精准搜索式
```
结构: "[核心词] | [受众/场景] [变化] 记，[结果吸引]"
示例: "眼膜 | 25岁黑眼圈困扰 3周逆袭记，终于不用遮瑕了"
特点: 直接命中搜索意图，信息密度高，易被搜索推荐
```

#### 方案 B — 解决方案式
```
结构: "[保姆级/全面] [主题] | [产品] [让我/帮我] [结果]"
示例: "保姆级黑眼圈护理指南 | 这款眼膜让我重获自信"
特点: 强调可信度和实用性，容易获得高互动
```

#### 方案 C — 悬念故事式
```
结构: "[对比转折] [产品], [秘密/关键词] [在这里|如下]"
示例: "我从不用遮瑕了，黑眼圈眼膜的秘密在这里 | 亲测有效"
特点: 制造悬念，提高点击率，容易引发评论区讨论
```

### 4. 关键词矩阵

**构成**：
```
| 关键词类型 | 数量 | 用途 | 示例 |
|----------|-----|------|------|
| 核心关键词 | 1-2 | 主要搜索流量 | 黑眼圈、眼膜补水 |
| 长尾关键词 | 2-3 | 精准用户 | 熬夜黑眼圈、淡化细纹、眼膜推荐 |
| 场景/人群词 | 2-3 | 细分人群 | 职场女性、熬夜党、护肤新手 |
```

**搜索流量预估**：
- 核心词: 高流量，竞争激烈
- 长尾词: 中等流量，精准用户
- 场景词: 低流量，高转化

## 关键风险管理

### 1. 禁用词汇（必须避免）

| 禁用类型 | 示例 | 替代表述 |
|---------|------|--------|
| **绝对词** | 最好、第一、唯一、顶级 | 深受喜爱、热门推荐 |
| **医疗暗示** | 修复、再生、激活、祛、治疗 | 改善、提亮、滋润、舒缓 |
| **夸大词** | 一夜见效、立竿见影、100%有效 | 几周、坚持使用、逐步改善 |
| **虚假背书** | 医生推荐、专家认可（无证据） | 用户推荐、好评率高 |

### 2. 高风险词汇（需加限定词）

| 高风险词 | 限定词方案 | 示例 |
|---------|----------|------|
| **有效** | 据说、据用户反馈、据不少人分享 | "据不少用户分享，效果不错" |
| **改善/缓解** | 可能、有助于、辅助、在正确使用下 | "坚持使用可能有助于改善" |
| **提升/增加** | 在一定程度上、相对而言 | "在一定程度上可以提升" |
| **适合** | 可能适合、大多数人、大部分人 | "一般适合各种肤质" |

### 3. 品类风险速查表

| 品类 | 风险级别 | 核心策略 | 禁忌 |
|------|---------|---------|------|
| **美妆类** | 中 | 强调修饰效果、快速见效 | 避免"永久"、"治疗" |
| **护肤类** | 中 | 强调真实测评、建立信任 | 避免医疗词、绝对词 |
| **3C/电子** | 低 | 强调使用场景、生活改变 | 避免虚假规格 |
| **食品/饮品** | 中 | 强调体验与氛围、品尝细节 | 避免健康功效声称 |
| **生活用品** | 低 | 强调品质提升、实用性 | 一般无特殊禁忌 |
| **教育/课程** | 中 | 强调学习成果、投资回报 | 避免保证成功 |
| **旅游/体验** | 低 | 强调沉浸式体验、值得去 | 避免虚假描述 |
| **母婴/儿童** | 高 | 强调安全性、舒适度、科学性 | 避免医疗声称，强调专业 |
| **健身/运动** | 中 | 强调运动效果、身体感受 | 避免"减肥"直接声称 |
| **医疗设备/美容仪器** | 超高 | 强调使用体验、非医疗属性 | 严禁医疗功效，强调"辅助" |
| **医美服务** | 超高 | 强调专业体验、安心感、社群反馈 | 严禁效果承诺，强调"改善感受" |
| **金融理财** | 超高 | 强调财商教育、风险意识 | 严禁收益承诺，披露风险 |
| **房产/家居** | 超高 | 强调居住体验、生活方式 | 严禁升值承诺，强调现实 |
| **宠物用品** | 低 | 强调毛孩子快乐、铲屎官省心 | 避免医疗声称 |
| **二手交易** | 低 | 强调发现惊喜、物品新生命 | 避免虚假品相描述 |

## 使用最佳实践

### 1. 信息采集检查清单

**必填项**（至少提供这 3 项）：
- [ ] 产品名称
- [ ] 核心功能/卖点（2-3 个最关键的）
- [ ] 目标用户画像（年龄段、职业、痛点）

**推荐补充**：
- [ ] 价格区间
- [ ] 初步内容角度（可选）
- [ ] 人设微调方向
- [ ] 禁忌/避免点
- [ ] 内容形式（图文 vs 视频）

### 2. 模式选择建议

```
信息完整度诊断流程:

输入信息 → 评估完整度
           ↓
        > 80%  → 直接完整模式 (5-10 分钟，强烈推荐)
        50-80% → 精简模式 (2-3 分钟) + 补全问卷
        < 50%  → 诊断模式 + 逐步补全

用户明确指定 → 按指定模式执行（用户需求优先）
```

### 3. 迭代与调整

如果初版输出不满意：
```
"我想要更 [文艺|干货|故事性] 的风格"
"能否调整标题方向到 [技术|情感|趣味] 方向"
"这个品类有没有特殊禁忌我应该注意"
"能给我一些爆款标题的数据支撑吗"
```

## 品类指导与案例库

### 高风险品类深度指导

#### 医美服务（超高风险）
```
策略要点:
- 强调专业体验（医生/设备专业）
- 强调安心感（卫生、安全、隐私）
- 引导社群反馈（真实案例、用户评价）

禁忌:
✗ "永久"、"彻底解决"、"去除"
```

## 快速参考表

### 何时使用 ohmyxhs

✅ **最适合场景**：
- 你有具体的产品要种草，但不知道怎么写
- 你想要多个标题和内容方案供选择
- 你担心踩小红书的违规红线
- 你想快速生成灵感，然后自己微调

❌ **不适合场景**：
- 你只想让我拍摄产品照片（技能外）
- 你想我直接帮你发布笔记（我只出方案）
- 你的产品是违法或有害的（合规优先）

### 使用建议

1. **提供详细的产品信息** — 信息越完整，输出质量越高（完整度 > 80% 效果最好）
2. **明确你的目标** — 是快速验证还是深度策划？选好模式（完整/精简/极速）
3. **告诉我你的约束** — 预算、品牌调性、禁忌词等
4. **反馈和迭代** — 不满意可以直接要求调整角度或重新生成

### 常见问题

**Q: 生成的内容能直接发布吗？**
A: 可以，但建议进行 2-3 轮调整：1) 合规检查 2) 品牌调性微调 3) 本地测试（小号发布）

**Q: 如何处理违规词汇？**
A: 1) 手动替换高风险词 2) 重新运行生成流程并明确指出禁忌词 3) 咨询平台规则

**Q: 标题能有多少个方案？**
A: 基础 3 个，但可以要求"再给 5 个标题创意"，通常能生成 8-10 个候选

**Q: 是否支持多国语言版本？**
A: 当前仅支持中文内容。跨国平台内容需要另行调整。
